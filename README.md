# UP-Basic

## Repository Description

This repository contains the documentation and analysis for a **web and mobile application extension** designed to support **new students at Universidad Pontificia Bolivariana (UPB)**.

The project focuses on easing the adaptation process of new students by centralizing useful tools and information, such as:
- A campus map for navigation.
- A help forum for student questions.
- Advisory and tutoring schedules.

The repository includes the analysis phase documentation, functional and non-functional requirements, and navigation structure of the project.
